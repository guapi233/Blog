# 删除排序链表中的重复元素

[题目编号-83](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-list/)



## 题目描述

给定一个排序链表，删除所有重复的元素，使得每个元素只出现一次。

**示例 1:**

```
输入: 1->1->2
输出: 1->2
```

**示例 2:**

```
输入: 1->1->2->3->3
输出: 1->2->3
```



## 解题思路

### 遍历

由于是已经排序好的链表，所以只需要在便利时判断前后两个节点是否一致，如果一致直接丢掉后一个节点即可

* 时间复杂度： O(n)
* 空间复杂度：O(1)
