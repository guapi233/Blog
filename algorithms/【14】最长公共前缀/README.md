# 最长公共前缀

[题目编号-14](https://leetcode-cn.com/problems/longest-common-prefix/)



## 题目描述

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1:

```
输入: ["flower","flow","flight"]
输出: "fl"
```

示例 2:

```
输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。
```

说明:

所有输入只包含小写字母 a-z 。



## 答案

### 遍历比较

将数组中的第一个元素作为标准，依次将其每一位与其它元素的相同位做对比

* 时间复杂度：O(n²)
* 空间复杂度：O(1)
