# 分隔链表

[题目编号-86](https://leetcode-cn.com/problems/partition-list/)



## 题目描述

给你一个链表和一个特定值 `x` ，请你对链表进行分隔，使得所有小于 `x` 的节点都出现在大于或等于 `x` 的节点之前。

你应当保留两个分区中每个节点的初始相对位置。

 

**示例：**

```
输入：head = 1->4->3->2->5->2, x = 3
输出：1->2->2->4->3->5
```



## 解题思路

### 双指针

分别维护两个指针，一个指向最后一个小值，一个指向最后一个大值，最后将将两个链表拼接起来即可

* 时间复杂度：O(n)
* 空间复杂度：O(1)
